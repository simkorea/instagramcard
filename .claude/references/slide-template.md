# 슬라이드 구성 템플릿

## 구성 원칙
- 전체 6~8장 고정
- 커버 1장, 본문 4~6장, 마무리 1장
- 각 슬라이드는 하나의 메시지에 집중
- **텍스트만 있는 슬라이드 금지 — 사진 또는 인포그래픽 반드시 포함**

---

## 슬라이드 유형 및 필수 시각 요소

| 장 | 유형 | 필수 시각 요소 |
|----|------|-------------|
| 1 | 커버 | Unsplash Photo Hero (full bleed) + Stat Card Row + Arrow Flow |
| 2 | 데이터/현황 | Photo Banner (상단 띠) + Bar Chart 또는 Stat Hero |
| 3 | 타임라인/절차 | Timeline 인포그래픽 + D-day 카운트다운 박스 |
| 4 | 경고/함정 | Photo Banner (경고 분위기) + Warning Icon Row |
| 5 | 비교/기회 | Bar Chart + Ring Chart (SVG 도넛) |
| 6 | 체크리스트 | Photo Banner + SVG Check Icon + Progress Bar |
| 7 | CTA 마무리 | Unsplash Photo Hero (full bleed) + Summary Card Row |
| * | 단지/상품분석 | Photo Banner 또는 다크bg + Spec Grid + Category Badge Row + Ratio Bar |
| * | 비교표 | Compare Column Table + Arrow Flow |
| * | 일정/날짜비교 | Date Schedule Row + Bar Chart |
| * | 체크리스트 ok/warn | Photo Banner + Checklist ok/warn (번호 배지 두 가지 색상) |

> 슬라이드 수·순서는 콘텐츠에 따라 조정 가능. 유형은 재사용 가능. `*` 는 선택적 추가 유형.

---

## 권장 슬라이드 구조

### 1장 — 커버
- 제목 (최대 20자)
- 서브카피 (최대 30자)
- 핵심 수치 3개 Stat Card
- 전체 흐름 Arrow Flow

### 2장 — 상황/현황
- 소제목 (최대 20자)
- Photo Banner (Unsplash, 주제 관련 사진)
- Bar Chart 또는 Stat Hero (핵심 숫자)
- 정보 리스트 3개
- 강조 문구

### 3장 — 타임라인/절차
- 소제목 (최대 20자)
- D-day 카운트다운 박스
- Timeline 인포그래픽 (4단계)
- 조건 카드 리스트
- 강조 문구

### 4장 — 경고/함정
- 소제목 (최대 20자)
- Photo Banner (경고 분위기, 적색 오버레이)
- Warning Icon Row (3종)
- 번호 배지 + 함정 카드 3개
- 경고 강조 문구 (적색)

### 5장 — 기회/비교
- 소제목 (최대 20자)
- Bar Chart (비교 데이터, CSS)
- Ring Chart SVG (비율/퍼센트)
- 기회 리스트 카드
- 강조 문구

### 6장 — 체크리스트
- 소제목 (최대 20자)
- Photo Banner (서류/계획 사진)
- Progress Bar
- SVG Check Icon + 체크 항목 5개
- 결과 박스 (Primary Blue 배경)

### 7장 — CTA 마무리
- Unsplash Photo Hero (full bleed)
- Summary Card Row 4개 (핵심 요약)
- 메인 CTA 문구
- 저장 버튼 + 공유 버튼 + 프로필 버튼
- 계정명

## 인포그래픽 선택 가이드

| 콘텐츠 | 사용 컴포넌트 |
|--------|-------------|
| 핵심 숫자 1개 강조 | Stat Hero |
| 두 값 비교 | Bar Chart (CSS) |
| 비율/퍼센트 | Ring Chart (SVG) |
| 단계·날짜 흐름 | Timeline |
| 경고 항목 요약 | Warning Icon Row |
| 완료 여부 목록 | SVG Check Icon + Progress Bar |
| 전체 요약 | Summary Card Row |
| A → B 변환 | Arrow Flow |

---

### 8장 — 단지/상품 분석형 (선택적 추가 유형)
- 소제목 (최대 20자)
- D-Day Badge (청약일·마감일)
- Spec Grid 2×2 (규모·가격·교통·입지)
- Category Badge Row (노선·카테고리)
- Ratio Bar (추첨/가점 비율)
- 강조 문구

> 청약 단지, 금융 상품, 강의 등 스펙 비교가 필요한 슬라이드에 활용

### 9장 — 비교표형 (선택적 추가 유형)
- 소제목 (최대 20자)
- Compare Column Table (A vs B 2열)
- Arrow Flow (전략 흐름)
- 인사이트 리스트 3개
- 강조 문구

> 두 가지 선택지를 명확하게 비교할 때 활용 (추첨 vs 가점, 공공 vs 민간 등)

---

## 예시 (양도세 급매물 카드뉴스, 7장)

- 슬라이드 1: "지금 급매물 사면 될까요?" (커버, Photo Hero + Stat Card)
- 슬라이드 2: "지금 서울에 무슨 일이?" (Photo Banner + Bar Chart)
- 슬라이드 3: "오늘이 사실상 마지노선" (D-day + Timeline)
- 슬라이드 4: "급매물이라도 피해야 할 것" (Photo Banner + Warning Icon)
- 슬라이드 5: "그래도 지금이 기회인 이유" (Bar Chart + Ring Chart)
- 슬라이드 6: "5초 안에 체크하세요" (Photo Banner + SVG Check + Progress Bar)
- 슬라이드 7: "저장해두고 꺼내보세요" (Photo Hero + Summary Card)
